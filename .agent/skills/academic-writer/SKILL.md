# Academic Writer (学术写作)

> 理解目标期刊 → 按 IMRaD 构建 → 精准表述

---

## 工作流程

```
写作任务 → 确认目标期刊（Nature/PRL/ACS Nano 等）
         → 按 IMRaD 搭骨架
         → 逐节填充
         → 自查清单
```

**核心原则**：
- 学术论文的目的是**传递证据链**，不是"吸引注意力"
- 每句话要么陈述事实，要么引用文献，要么推理论证
- 简洁 > 花哨。能一句话说清的不用两句

---

## IMRaD 框架

### Abstract（三段式）
```
[Background] X is important because... However, Y remains poorly understood.
[Approach & Finding] Here we demonstrate/show/report that Z, using [method].
[Impact] Our results suggest/establish/provide... for [application/field].
```
- 必须包含关键定量结果（数值 + 误差）
- 字数：一般 150-250 词

### Introduction（漏斗结构）
```
¶1: 大背景（领域重要性）
¶2: 已有工作（文献综述）
¶3: Knowledge Gap（知识空白）→ "However, ... remains unclear"
¶4: 本文贡献（"Here we demonstrate..."）
```

### Methods / Experimental
- 足够详细以支持重复实验
- 物理常数必须注明出处
- 仪器型号、参数、厂商

### Results
- 每段以关键发现开头（topic sentence）
- 图表先于文字（读者先看图）
- 数据呈现与解释分离

### Discussion
- Results 说 "what"，Discussion 说 "why" 和 "so what"
- 必须讨论：局限性、alternative explanations、未来方向

### Conclusion
- 不引入新数据或新论点
- 简明重述核心发现和影响

## 术语规范

| ❌ 避免 | ✅ 使用 | 原因 |
| :--- | :--- | :--- |
| "查阅了数据" | "Data were taken from [Ref]" | 口语 vs 学术 |
| "自发演化" | "The algorithm converged to" | 误导性 |
| "完美匹配" | "closely matches / is in good agreement with" | 过于绝对 |
| "无衍射" | "quasi-non-diffracting / Bessel-like" | 理想化 |
| "众所周知" | "[具体引用]" | 需出处 |
| "..." (公式省略) | [完整公式] 或 "see Ref [X]" | 不可省略 |

---

## 公式写作规范

1. **不允许省略号**: 公式必须完整，或引用文献
2. **变量首次出现必须定义**: "$d_p$ is the penetration depth"
3. **单位**: 使用 SI 单位，数值与单位间加空格 (532 nm, not 532nm)
4. **近似条件**: 明确写出 "under the assumption that..."

---

## 引用规范

- 关键 claim 必须有引用
- 物理常数引用原始测量论文
- 避免过度自引（< 20%）
- 引用格式随目标期刊（Nature: 上标数字, PRL: 方括号数字）

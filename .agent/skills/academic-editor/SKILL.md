# Academic Editor (学术编辑)

> 逻辑 → 结构 → 段落 → 句子 → 词语 → 格式，六层递进

---

## 工作流程

```
草稿 → 逻辑层（论证链完整性）
     → 结构层（IMRaD 合规性）
     → 段落层（每段一个核心观点）
     → 句子层（清晰度、主动语态）
     → 词语层（精准用词、限定词）
     → 格式层（引用、公式编号、图表标签）
```

---

## 六层编辑法

### 第零层：逻辑编辑 (Logic Editing) 🆕
**关注**：论证链是否完整、是否存在逻辑谬误

| 检查项 | 示例 |
| :--- | :--- |
| 因果跳跃 | "NA高 → 信号强"（缺中间推理） |
| 循环论证 | "信号强因为NA好，NA好因为信号强" |
| 遗漏竞争假说 | 只讨论利好因素，不讨论反面 |
| Claim vs Evidence 失衡 | 大 Claim 配小 Evidence |

### 第一层：结构编辑
同原 Super Editor，但增加 IMRaD 合规检查。

### 第二层：段落编辑
同原 Super Editor。

### 第三层：句子编辑
同原 Super Editor，额外关注：
- 被动 vs 主动语态（Methods 用被动，Results 用主动）
- 时态一致性（Methods: past tense, Discussion: present tense）

### 第四层：词语编辑
同原 Super Editor，额外关注限定词：
- "perfect" → "near-perfect"
- "proves" → "suggests / demonstrates"
- "novel" → 少用，让审稿人判断

### 第五层：格式编辑 (Format Editing) 🆕
**关注**：引用格式、公式编号、图表标签、SI 单位

| 检查项 | 规范 |
| :--- | :--- |
| 公式 | 关键公式须编号 (Eq. 1) |
| 图表 | 所有图表须在正文中引用 |
| 引用 | 格式统一，出处完整 |
| 单位 | SI 制，数值与单位间加空格 |
| 缩写 | 首次出现写全称 |
